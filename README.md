# 📊 Call Center Performance Analysis Dashboard

## 📌 Project Overview

This project is an interactive Excel dashboard developed to analyze Call Center performance data. The dashboard provides insights into call volume, agAent performance, customer satisfaction, response times, and issue resolution trends through dynamic visualizations and interactive reports.

## 🎯 Objectives

* Analyze overall call center performance.
* Monitor inbound and outbound call trends.
* Evaluate agent productivity and efficiency.
* Analyze customer satisfaction ratings.
* Track issue resolution and response times.
* Identify peak call periods and performance patterns.

## 🛠️ Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers and Timelines
* Excel Formulas and Functions
* Conditional Formatting

## 📈 Dashboard Features

* KPI Cards for Total Calls, Resolved Calls, Unresolved Calls, and Average Customer Satisfaction
* Call Volume Analysis by Date and Time
* Agent-wise Performance Analysis
* Customer Satisfaction Rating Analysis
* Issue Resolution and Response Time Analysis
* Call Status Distribution (Resolved vs Unresolved)
* Interactive Filters and Slicers

## 📊 Key Insights

* Peak call volumes occur during specific days and time periods.
* Some agents consistently achieve higher resolution rates and customer satisfaction scores.
* Response times directly impact customer satisfaction levels.
* The majority of calls are resolved within the target timeframe.
* Performance analysis helps identify opportunities for process improvement and resource optimization.

## 🚀 How to Use

1. Download the `.xlsx` file.
2. Open it using Microsoft Excel.
3. Refresh the Pivot Tables if required.
4. Use slicers and filters to interact with the dashboard and explore insights.
